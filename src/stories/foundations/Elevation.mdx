# Elevation

Elevation communicates visual layering and hierarchy.

| Token | Typical usage |
|---|---|
| `shadow.sm` | Subtle control or surface separation |
| `shadow.md` | Cards and floating elements |
| `shadow.lg` | Drawers, modals, high-priority overlays |

## Guidance

Use elevation to communicate layer hierarchy. Avoid adding shadows only for decoration.

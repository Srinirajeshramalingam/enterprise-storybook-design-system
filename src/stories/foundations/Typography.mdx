# Typography

The typography system creates hierarchy and readability across products and CMS experiences.

## Font family

**Nunito Sans** with system fallbacks.

## Type scale

| Token | Size | Typical usage |
|---|---:|---|
| `font.size.xs` | 12px | Metadata, helper text |
| `font.size.sm` | 14px | Body small, controls |
| `font.size.md` | 16px | Default body |
| `font.size.lg` | 20px | Section heading |
| `font.size.xl` | 24px | Component / page heading |
| `font.size.display` | 48px+ | Hero heading |

## Guidance

Typography should communicate information hierarchy, not create decoration without purpose.

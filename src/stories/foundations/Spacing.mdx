# Spacing

A consistent spacing scale creates predictable rhythm across components and layouts.

## Base scale

| Token | Value |
|---|---:|
| `space.1` | 4px |
| `space.2` | 8px |
| `space.3` | 12px |
| `space.4` | 16px |
| `space.5` | 20px |
| `space.6` | 24px |
| `space.8` | 32px |
| `space.10` | 40px |
| `space.12` | 48px |

## Guidance

Prefer the defined spacing scale over arbitrary one-off values.

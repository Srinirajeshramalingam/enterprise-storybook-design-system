# Color

Color tokens communicate hierarchy, interaction, feedback, and brand identity.

## Core palette

| Token | Value | Usage |
|---|---|---|
| `color.brand.primary` | `#555AB9` | Primary actions |
| `color.text.primary` | `#111827` | Main text |
| `color.text.secondary` | `#64748B` | Supporting text |
| `color.border.default` | `#CBD5E1` | Default borders |
| `color.surface.default` | `#FFFFFF` | Primary surface |
| `color.surface.subtle` | `#F5F7FA` | Secondary surface |
| `color.success` | `#166534` | Success feedback |
| `color.warning` | `#92400E` | Warning feedback |
| `color.danger` | `#991B1B` | Error / danger feedback |

## Guidance

Use semantic tokens based on purpose rather than choosing raw colors independently inside each component.

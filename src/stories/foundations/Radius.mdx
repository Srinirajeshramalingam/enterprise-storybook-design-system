# Radius

Radius tokens create consistent shape language across controls, surfaces, and overlays.

| Token | Value | Usage |
|---|---:|---|
| `radius.sm` | 6px | Compact controls |
| `radius.md` | 10px | Inputs and controls |
| `radius.lg` | 16px | Cards and surfaces |
| `radius.xl` | 24px | Large containers |
| `radius.full` | 999px | Pills and circular controls |
